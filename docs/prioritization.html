Conservation Prioritization

Population Status and Scenario Explorer

Published

March 26, 2026

Overview

This page presents a demographic status assessment for all 80 local bull trout populations across 26 core areas in the Columbia Headwaters Recovery Unit. All estimates are derived from a hierarchical state-space Ricker model fitted to annual redd count time series (Model v1_22 / Data v1_10).

The assessment addresses two management questions:

  1. Where are populations headed? — The intrinsic growth rate (λ) estimated under recent covariate conditions reveals which populations are growing, stable, or declining.
  2. Which threats matter most, and for which populations? — Sensitivity scenarios quantify how much λ would improve if a single threat lever were reduced while all others remain at current levels, highlighting where management investment would have the greatest return.

Key Findings

Population-Level Summary
  • 64 of 80 populations (80%) have λ < 1 under recent conditions, indicating ongoing decline.
  • λ ranges from 0.81 (most rapidly declining) to 1.12 (most rapidly growing) across all populations.
  • 55 populations have ≥ 50% probability of dropping below 20 adults at some point in the next 7 years (one bull trout generation).
  • Every population carries some demographic risk over the projection horizon — underscoring the recovery unit-wide urgency of threat management.

Column guide:

Column group What it shows
Current Status Median λ with 90% credible interval under last-observed covariate conditions. λ > 1 = growing; λ < 1 = declining.
Demographic Risk Probability of the latent adult population falling below a threshold at any point in a 7-year forward projection (mean environmental conditions).
Sensitivity Median λ if a single threat lever is improved to the level shown, with all other covariates held at current. These highlight the biggest single-lever opportunity per population.

Table sorted by ascending λ by default (most declining populations first).

Scenario Explorer

The interactive scenario tool extends the sensitivity analysis by letting you set any combination of covariate conditions simultaneously — useful for evaluating management packages that address multiple threats at once, and for identifying which populations respond most strongly to each lever.

The tool provides three views:

  • λ Comparison — bar chart of median λ per population under current vs. scenario conditions, coloured by core area
  • Risk Biplot — scatter plot of intrinsic growth rate (b₀ᵣ) vs. probability of local population loss; bubble size reflects habitat extent; a useful prioritization framework combining population trajectory and demographic vulnerability
  • Data Table — the full status table with scenario-adjusted values, downloadable as CSV

Methods Note

Demographic status is estimated from the hierarchical Ricker state-space model described on the Model Results page. Briefly:

  • Process model — log-abundance follows an AR(1) Ricker equation with population-level random slopes for flow, summer temperature, invasive fish proportion, and lake trout proportion (lake-connected cores only).
  • Observation model — annual redd counts are NB2-distributed given latent abundance, scaled by survey-section detection and a contemporaneous flow effect on observer efficiency.
  • λ estimation — intrinsic growth rate at each population is computed from 1,000 marginal posterior draws (independent Normal approximations centred on MAP estimates), providing the median and 90% credible interval shown in the table.
  • Demographic risk projection — 7-year forward simulations using MAP state estimates as initial conditions and AR(1) process noise drawn from the posterior. No management intervention is assumed. Results are reported as the probability that latent adult abundance falls below a threshold at any point during the projection horizon.
