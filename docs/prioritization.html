Overview

This page presents the demographic status of all 80 local bull trout populations across 26 core areas in the Columbia Headwaters Recovery Unit. Estimates are derived from a hierarchical state-space Ricker model fitted to annual redd count time series (Model v1_22 / Data v1_8).

Interpreting the table:

  • λ (median) — intrinsic growth rate under current covariate conditions (last observed year). Values > 1 indicate a growing population; < 1 declining.
  • Qext (N < 20) — probability of dropping below 20 adults at any point in a 7-year forward projection under mean environmental conditions.
  • Sensitivity columns — λ if a single threat lever is improved while all others remain at current levels. These highlight which interventions offer the greatest potential benefit per population.

Interactive Scenario Tool

The scenario tool lets you set covariate conditions across all populations simultaneously and see how projected λ changes — useful for comparing the relative benefit of different management actions and identifying which populations respond most strongly to each threat lever.
